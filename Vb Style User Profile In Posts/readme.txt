Vb Style User Profile In Posts

This will display the user profile in topics in a more organised look and will be make posts on your forum to look more beautiful.

One can select the look from the setting available in 
Admin panel » Configuration » Current Theme » Set VB Style Effect In Posts (for Poster area).

Admin can choose which style they want by choosing curve or plain styles from drop-down boxes.
